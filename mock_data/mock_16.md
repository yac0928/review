# 自傳

## 個人簡介

我叫姚柔安，來自國立清華大學資訊工程學系。從小，我就對電腦世界充滿好奇，總是被那些複雜的程式碼和系統運作原理深深吸引。在清華資工的四年大學生涯中，我努力學習資訊領域的基礎知識與核心技術，並逐步將學習重心放在我最感興趣的資訊安全與加密技術上。我的學業成績雖然稱不上頂尖，但修課平均仍維持在 3.69，顯示我在學習上的投入與基本掌握。

## 求學經歷與學習動機

大學四年，我嚴謹地完成了系上所要求的所有核心課程，像是「資料結構」、「演算法」、「計算機組織」和「作業系統」。這些課程為我打下了堅實的程式設計與系統知識基礎，讓我對電腦科學的廣度有了全面的認識。我特別享受那些需要邏輯思考和問題解決的挑戰，像是演算法課程中優化時間複雜度的練習，或是作業系統專題中實作記憶體管理模組。這些經驗不僅提升了我的程式能力，也培養了我面對複雜問題時的耐心與細心。

大三開始，我對於資訊安全的領域產生了濃厚的興趣，尤其是加密技術如何保障資訊傳輸的隱私與完整性。我發現這是一個既具挑戰性又充滿實際應用價值的領域。儘管系上沒有專門的資安研究室經驗，我仍透過修習相關課程和自主學習，努力在這方面耕耘。我深刻體會到，資安防護的環節環環相扣，任何一處的疏忽都可能導致嚴重的後果。這種責任感也促使我希望能更深入地學習。

## 資訊安全領域的探索與實作

在大學期間，我主要透過課程專題和個人自學專案來探索資訊安全與加密技術。例如，在「計算機網路」課程的期末專題中，我與組員們一同設計並實作了一個簡易的加密通訊系統。我們使用了 Python 語言，並嘗試實作了 AES 加密演算法來保護傳輸的文字訊息，同時利用 Socket Programming 建立了點對點的通訊連線。雖然系統功能相對基礎，但這個專案讓我第一次親手將抽象的加密理論轉化為實際可運作的程式碼，並理解到網路傳輸安全的重要性，以及如何防範中間人攻擊的基本概念。這段經驗讓我對對稱式加密有了初步的認識，也學到了如何處理資料序列化與反序列化。

另一個讓我有深刻印象的專案是在「網路安全」課程中，我們被要求分析常見的網路攻擊手法。我選擇了針對 SQL Injection 進行研究，並模擬攻擊情境，探討如何透過程式碼修改來避免這類漏洞。我閱讀了 OWASP Top 10 的相關內容，學習到許多網站安全開發的實務建議。透過實際撰寫一個易受攻擊的網頁應用程式，再逐步修復其安全漏洞，我了解到輸入驗證和參數化查詢對於防止 SQL Injection 的關鍵作用。這個過程讓我對網路安全有了更具體的理解，也發現資訊安全並非單純的技術問題，更是需要綜合考量系統設計、開發流程以及使用者習慣的複雜挑戰。

除了課程專題外，我也會利用課餘時間自學一些資安工具和知識，例如透過 CTF (Capture The Flag) 平台練習一些基本的逆向工程、網頁漏洞分析和密碼學挑戰，嘗試使用 Linux 環境下的工具如 `Wireshark` 進行封包分析，以理解網路協定運作與潛在的安全風險。這些零散的學習雖然不夠系統，但卻讓我保持對資安領域的好奇心與熱情，並初步培養了解決資安問題的直覺。

## 課外活動與個人特質

除了學業之外，我也積極參與了大學合唱團，並在大三時擔任了領唱的職務。作為領唱，我需要協調不同聲部的練習進度，確保大家能夠和諧地合作，共同呈現出最完美的演出。這份經驗讓我學會了如何在團隊中有效溝通、傾聽他人的意見，並在壓力下保持冷靜，帶領團隊克服困難。合唱團的每一次演出都讓我體驗到團結合作的重要性，這與資工領域中團隊開發專案的需求不謀而合。我相信這些軟實力對於未來在學術研究或業界合作上都將是寶貴的資產。

## 申請動機與未來展望

我對資訊工程的熱情與對資訊安全的興趣，促使我希望能繼續深造。交大資工所在國內外都享有卓越的聲譽，特別是在資訊安全領域的研究成果，更是我嚮往的目標。我深信在貴所的豐富資源與優質學習環境下，我能更系統性地學習資安領域的先進知識，並深入參與前瞻性的研究。我期許自己能在碩士班期間，不僅強化專業技能，更能培養獨立思考與解決複雜問題的能力。雖然目前的研究方向還不是非常明確，但渴望在資安領域貢獻一份心力、成為資安領域的一份子的熱情，驅使我邁向這一步。希望有機會進入貴所就讀，為我的資安專業之路開啟新的篇章。

# 讀書計畫

## 短期目標 (入學前至碩一上學期)

*   **基礎知識鞏固與預備：**
    *   複習大學期間的資訊工程核心課程，特別是「資料結構」、「演算法」與「計算機網路」等，確保基礎穩固。
    *   加強 Python 與 C/C++ 程式設計能力，熟悉常用的函式庫與工具。
    *   閱讀資訊安全領域的入門書籍與經典論文，對常見的攻擊手法、防禦機制與加密技術有更全面的認識。
    *   主動了解交大資工所資安相關實驗室的研究方向與教授專長，初步探索未來可能的指導教授人選。
*   **課程修習與學術適應：**
    *   規劃修習「高等密碼學」、「網路安全」、「作業系統安全」等核心資安課程，深入學習密碼學原理與網路防禦機制。
    *   旁聽或選修一些與資訊安全相關的選修課程，例如「區塊鏈技術」或「惡意程式分析」，擴展知識廣度。
    *   積極參與實驗室討論、研討會與專題演講，學習研究方法，並與學長姐及教授交流，了解學術界的研究前沿。
    *   逐步適應碩士班的研究型學習模式，提升文獻閱讀、分析與報告能力。

## 中期目標 (碩一下學期至碩二上學期)

*   **研究方向聚焦與專題實作：**
    *   確定具體的研究領域，例如軟體安全、密碼應用、網路攻防或區塊鏈安全等，並在指導教授的協助下選定碩士論文題目。
    *   參與實驗室的研究專案，學習如何在實際問題中應用所學知識，並從中培養解決問題的能力。
    *   若有機會，將積極尋找與研究領域相關的校外實習機會，將理論知識與業界實務結合，累積實戰經驗。
    *   開始撰寫碩士論文初稿，並定期與指導教授討論研究進度，接受意見回饋並持續改進。
*   **技術能力深化與論文撰寫：**
    *   針對論文題目所需的專業技能進行強化，例如學習新的程式語言、開發框架或分析工具。
    *   深入閱讀與論文相關的學術論文與研究報告，了解最新的研究進展，並探索創新的研究方向。
    *   積極參與學術研討會或國內外會議，發表研究成果，並從同行交流中獲得啟發與建議。
    *   在教授指導下，持續完善論文內容，確保研究成果的深度與廣度。

## 長期目標 (畢業後)

*   **貢獻資訊安全領域：**
    *   希望未來能投入資訊安全相關的產業或研究單位，成為一名專業的資安工程師或研究員，為保護數位世界貢獻心力。
    *   持續學習與自我提升，追蹤資訊安全領域的最新發展與技術趨勢，不斷更新知識技能。
    *   若有機會，也希望能參與開源專案或社群，與同行分享知識與經驗，共同推動資安技術的進步。
    *   期許自己能在特定資安領域中深耕，成為該領域的專家，並能為社會帶來正面的影響。
