## 自傳

我叫陳煦陽，來自台灣大學電機工程學系。從電機系踏入資工領域，對我而言，是一段探索與發掘熱情的旅程。大學四年來，我的學業成績雖然是修課平均 3.33，但這過程中，我學習了許多電機工程的紮實基礎，也逐漸將興趣轉向資訊工程，特別是對人工智慧與機器學習領域充滿好奇與熱情。我相信我的跨域背景，能為交大資工所帶來不同的視角與能量。

**跨域學習的動機與轉折**
大二時，在修習「線性代數」與「機率」等基礎課程的過程中，我開始對背後的數學邏輯與資料分析產生濃厚的興趣。當時，電機系課程多聚焦於硬體電路設計與訊號處理，雖然也很扎實，但我心底深處對「如何讓機器思考、學習」這件事更為著迷。一個重要的轉折點發生在大三上學期，我選修了外系的「機器學習導論」課程。這門課讓我對監督式學習與非監督式學習的概念有了初步認識，並第一次實際操作了 Python 搭配 Scikit-learn 進行數據預測。雖然只是入門，但那種將冰冷的數據轉化為有意義洞察的感覺，徹底點燃了我對軟體與人工智慧的熱情。自此之後，我便積極地投入自學與尋找相關資源，希望彌補我在純資工背景上的不足。

### 研究經歷
為了更深入理解機器學習的應用，大三暑假，我進入了系上陳教授的「智慧感知與計算實驗室」參與專案。我們當時的研究主題是「基於深度學習的心電圖異常訊號辨識系統」。我的主要職責是協助資料前處理與模型訓練。我們使用 Python 搭配 NumPy 與 Pandas 清理了大量的醫療數據，並利用 TensorFlow 框架搭建了卷積神經網絡 (CNN) 模型來識別心律不整等異常模式。這個專案讓我首次體驗到從真實數據中提取特徵、設計模型並評估效能的完整流程。過程中，我遇到了資料標註不一致、模型過擬合等挑戰，在教授的指導下，我學習了如何應用交叉驗證、Dropout 等技巧來優化模型表現，這讓我對深度學習的實務應用有了更具體的認識，也讓我了解到研究過程需要不斷嘗試與迭代。

### 業界實習與國際志工
在研究室的經驗之後，我意識到除了學術研究，業界的實踐也同樣重要。大四上學期，我獲得了在一家新創科技公司「智連創新」的軟體開發實習機會。我在資料分析團隊中，主要負責協助開發一個用於電商推薦系統的初步原型。我使用了 Python 處理使用者行為日誌，並嘗試運用協同過濾 (Collaborative Filtering) 演算法來產生商品推薦。雖然實習時間不長，但這次經驗讓我接觸到真實的企業數據與開發流程，也讓我對軟體開發的協作模式有了初步了解。

除了學術與實習，我也十分重視社會回饋。大二那年，我參加了「非洲數位教育國際志工」計畫，前往迦納的一個偏鄉地區，協助當地學校架設基礎電腦教室，並教導孩子們基礎的電腦操作與簡單的程式概念。雖然當地資源有限，但我發現透過簡單的Scratch教學，孩子們也能展現出對數位世界的濃厚興趣。這次經驗讓我看到資訊科技對於弭平數位落差、提升教育水平的巨大潛力，也更堅定了我未來希望將所學應用於對社會有意義的領域。

總結來說，我的電機背景給了我紮實的數理基礎，而跨域學習的歷程、實驗室與業界的實作經驗，以及國際志工的服務經歷，都讓我對資工領域的潛力與應用充滿了想像。我渴望在交通大學資訊工程學系的優質環境中，繼續深耕人工智慧與機器學習，將我在電機系所學的理論知識與跨域視野，與資工的專業技術結合，為未來的研究與應用貢獻一份心力。

## 讀書計畫

進入交大資工所是我的重要目標，我希望透過研究所的深造，系統性地學習資訊工程的專業知識，尤其是在人工智慧與機器學習領域。我期待能將過去在電機系累積的數理基礎與思維邏輯，有效地融入資工領域，開創更廣闊的學習視野。
