# 曹靖雯 申請國立交通大學資訊工程學系碩士班 推甄文件

## 自傳

我是曹靖雯，目前就讀於中興大學資訊工程學系。從大學時期開始，我便對低階系統操作、韌體安全以及嵌入式系統的硬體交互作用懷抱著濃厚的興趣。我在學業上力求表現，大學至今的學業成績維持在 3.85/4.00，在系上名列前茅，這不僅是我對知識追求的展現，也為我奠定扎實的資訊工程基礎。我深信，正是這些基礎訓練與我對硬體與安全領域的熱情，促使我決心投入交大資工所，繼續探索嵌入式系統的無限可能與其背後的資安挑戰。

### 學術背景與研究動機

我在中興大學資工系的課程學習中，不斷深化對計算機科學的理解。大二時修習「計算機組織」與「數位邏輯設計」時，我對CPU如何執行指令、記憶體如何存取資料等底層機制產生了強烈好奇。隨後在「作業系統」與「嵌入式系統」等課程中，更進一步接觸到韌體如何與硬體協同運作、管理系統資源的細節。特別是課程中提到嵌入式系統因資源限制與部署環境的特殊性，導致其面臨獨特的資安威脅，例如硬體特洛伊木馬（Hardware Trojan）攻擊，這徹底點燃了我對這個領域的研究熱情。我意識到，要確保這些無處不在的智慧裝置安全可靠，必須從韌體與硬體的結合點下手。

### 研究經歷與實務專案

為了將理論知識付諸實踐，我積極投入實驗室與專題研究，累積了寶貴的實務經驗。

-   **基於功耗側通道分析的硬體特洛伊木馬偵測**
    大三下學期起，我加入了實驗室（虛構名稱：嵌入式系統安全實驗室），在指導教授帶領下，參與了為期一年的「基於功耗側通道分析的硬體特洛伊木馬偵測」研究專案。
    -   **專案目標：** 旨在開發一種非侵入式方法，透過分析嵌入式裝置的功耗側通道資訊，偵測潛藏於硬體電路中的惡意特洛伊木馬。
    -   **我的職責：** 主要負責功耗數據採集系統的建置、數據前處理演算法的開發，以及機器學習模型的訓練與評估。
    -   **技術細節：**
        -   利用 **Keysight DSOX2004A 示波器** 搭配電流探棒，成功從 **STM32F4** 微控制器執行特定加密演算法（AES）的過程中，精準採集了約 **10,000 筆** 的功耗波形數據。
        -   在數據前處理階段，我使用 **Python 搭配 NumPy 與 SciPy 函式庫** 實現了平滑濾波、同步化與降噪等演算法，有效提升了數據品質。
        -   在特洛伊木馬偵測模型方面，我嘗試了多種機器學習演算法，最終選擇以 **scikit-learn 框架下的 Support Vector Machine (SVM)** 與 **TensorFlow 框架下的 Convolutional Neural Network (CNN)** 進行模型訓練。透過反覆調校參數，我們成功在模擬含有特洛伊木馬的資料集中，達到了 **92% 的偵測準確度**。
    -   **學到與反思：** 在這個專案中，我深刻體會到側通道資訊的複雜性與其在安全性分析上的巨大潛力。第一次意識到，即便是看似微小的功耗波動，背後都可能隱藏著惡意行為的蹤跡。專案讓我熟練掌握了嵌入式裝置的低階操作、數據採集與分析技能，並學會了如何將機器學習應用於資安問題。

-   **基於 FreeRTOS 的智慧家居閘道器韌體開發**
    在大三下的「嵌入式系統設計」課程專題中，我與組員共同完成了「基於 FreeRTOS 的智慧家居閘道器韌體開發」專案。
    -   **專案目標：** 設計並實作一個能連接多種智慧感測器（如溫濕度、光照）的物聯網閘道器，並透過韌體實現數據採集、本地處理與雲端通訊。
    -   **我的職責：** 負責 **ESP32 微控制器** 上的 **FreeRTOS 即時作業系統** 任務排程、感測器驅動程式開發以及 **MQTT** 協定通訊模組的實作。
    -   **技術細節：**
        -   利用 **C/C++** 語言開發韌體，成功整合了 DHT11 溫濕度感測器與 BH1750 光照感測器，並確保數據採集的即時性與穩定性。
        -   設計了多個 FreeRTOS 任務，包括感測器數據讀取任務、數據處理任務和 MQTT 雲端上傳任務，並透過佇列（Queue）實現任務間的同步通訊。
        -   實現了基於 **TLS 的 MQTT 安全連線**，確保數據傳輸的機密性與完整性。
    -   **學到與反思：** 這個專案讓我對即時作業系統的運作機制有了深入了解，尤其是在有限資源下進行多任務協調的挑戰。我第一次體會到，在資源受限的嵌入式環境中，韌體設計的效率與穩定性對整體系統效能至關重要，也意識到在 IoT 裝置上實作安全通訊的複雜性。

### 未來展望與申請動機

我的求學與研究經歷，讓我對韌體安全、嵌入式系統設計以及硬體特洛伊防護領域充滿了探索的渴望。國立交通大學資訊工程學系在這些領域的研究實力與豐沛資源，正是我追求更高學術目標的理想環境。交大資工所不僅擁有多位在資訊安全、嵌入式系統及計算機結構方面享譽國際的教授，其與產業的緊密連結及優良的學術氛圍，都讓我心嚮往之。我期望能進入交大資工所，在更具挑戰性的研究環境中，磨練我的專業技能，擴展我的學術視野，並最終為韌體安全領域貢獻一份心力。
