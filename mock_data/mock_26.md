### 學術背景與跨域思維的啟蒙

我在清華大學電機工程學系累積了學業成績 3.69 的表現。電機系的訓練讓我對數位邏輯、計算機結構、演算法及資料結構有了嚴謹的認識，特別是在「計算機概論」與「物件導向程式設計」這兩門課程中，我第一次體會到透過程式碼解決問題的巨大魅力。我發現，除了硬體層面的設計與優化，軟體定義的靈活性與廣闊應用空間更讓我著迷。

真正激發我轉向資訊安全領域的轉捩點，是大三上學期修習「網路程式設計」時，老師在課堂上提及的資安漏洞與攻擊手法。當時我們利用 Python 撰寫 Socket 程式，模擬了客戶端與伺服器間的通訊，但在探討 SQL Injection 或 Cross-Site Scripting (XSS) 等常見攻擊時，我才意識到即便精心設計的系統，也可能因微小的程式邏輯疏忽而面臨巨大風險。這種「攻防」的思維模式，以及對隱私、數據完整性的深層影響，讓我對資訊安全產生了濃厚的興趣。

從那時起，我便積極利用電機系的資源，並自主學習相關知識。我理解到電機工程的背景，讓我對底層硬體、網路協定有更直觀的認識，這對於理解資安漏洞的根源，並設計更 robust 的防禦機制，是寶貴的資產。例如，在學習密碼學時，我能更快地理解其在硬體實作上的考量；在網路安全中，也能從更底層的角度思考封包傳輸與協定漏洞。這種跨領域的視野，讓我能從宏觀到微觀，更全面地審視資安問題。

### 專業經歷與技能累積

儘管我的大學主修是電機工程，我積極透過課程專題、業界實習與個人專案來彌補並強化我的資工專業能力。

**課程專題：深度學習在惡意程式檢測中的應用**
在大三下的「機器學習」課程專題中，我與組員合作，探討了如何利用深度學習模型來檢測惡意程式碼。我們蒐集了約 15,000 個 PE 格式的良性與惡性執行檔樣本，並利用 Python 結合 pefile 函式庫提取了約 200 個特徵，包括匯入函式、API 調用、Section 資訊等。我們嘗試了多種模型，最終發現以 PyTorch 訓練的卷積神經網路 (CNN) 在特徵工程後，能達到約 92% 的準確度，優於傳統的 Random Forest (88%)。這個專題讓我第一次完整體驗到資料前處理、模型選擇、訓練優化與評估的流程，也讓我深刻認識到特徵工程對於模型效能的決定性影響。

**業界實習：SecureFlow Technologies 軟體開發實習生**
111 學年度暑假，我在一家專精於資安解決方案的初創公司 SecureFlow Technologies 擔任軟體開發實習生。為期兩個半月的實習期間，我主要參與一個內部工具的開發，該工具旨在自動化檢測網路設備的配置漏洞。我的主要職責包括：
1.  **功能開發：** 使用 Python 編寫腳本，整合 Nmap 掃描結果，並基於預設的安全規範（如 CIS Benchmarks）解析 SSH 和 Telnet 服務的配置檔，識別潛在弱點。我開發了針對特定網路設備品牌（Cisco, Juniper）的解析模組，有效識別了 10 種以上常見的配置錯誤。
2.  **資料庫設計：** 協助設計並實作 PostgreSQL 資料庫，儲存掃描結果與漏洞報告，確保數據的結構化與可查詢性。我負責設計了約 5 個資料表，並撰寫了 SQL 查詢語句以生成客製化報告。
3.  **單元測試：** 針對我負責的解析模組，撰寫超過 50 個單元測試案例，確保程式碼的穩定性與正確性，將潛在 bug 的檢出率提升了約 30%。
這段實習經驗讓我第一次將所學應用於真實的資安場景，從版本控制 (Git)、敏捷開發流程，到團隊協作，都讓我獲益匪淺。我第一次體會到軟體品質與系統測試的重要性，也對資安產品的開發流程有了更實際的理解。

**個人 Side Project：基於區塊鏈的去中心化檔案儲存系統原型**
由於對區塊鏈技術在資安領域的應用感到好奇，我在大四上學期利用課餘時間，獨立開發了一個去中心化檔案儲存系統的原型。我使用 Solidity 撰寫智慧合約，並部署在 Ganache 私有鏈上，負責管理檔案的元數據和存取權限。前端介面則使用 ReactJS 與 Web3.js 進行開發，讓使用者可以上傳、下載加密後的檔案。雖然這只是一個原型，但它讓我對區塊鏈的不可篡改性、去中心化身份驗證有了深入的實作經驗，也讓我意識到分佈式系統在提升數據安全與隱私方面的巨大潛力。
