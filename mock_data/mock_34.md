### **自傳**

我是高宜珊，目前就讀於國立交通大學資訊工程學系。從小，我對於解決問題就充滿熱情，尤其是透過邏輯思考來找出最佳解方，這也引導我選擇了資訊工程這條路。回顧我的大學四年，雖然我的成長之路或許與許多同學有所不同，但我深信這些經歷都讓我成為一個更為堅韌且目標明確的人。

我的大學生活，很大一部分時間都圍繞在平衡學業與打工之間。由於家庭經濟的考量，我從大一開始就必須利用課餘時間打工來支付學費和生活費。我先後在校園附近的咖啡廳擔任服務生，週末則在便利商店值大夜班，大三開始也接了幾份家教。這些非技術性的工作，看似與我的專業無關，卻磨練了我許多寶貴的軟實力，例如時間管理、與不同類型顧客溝通的技巧，以及在壓力下保持效率的能力。我學會了如何在一週內安排好十幾個小時的工時，同時確保不會犧牲課業表現。大學四年下來，我的學業成績雖未特別突出，修課平均維持在 3.23 左右，但這是在我同時負擔學費與生活費的情況下，努力不懈的成果，也反映了我對資訊領域的堅持與熱情。

儘管如此，我從未放棄對資訊工程專業知識的追求。在系上，我對程式語言、編譯器與軟體工程等課程特別感興趣。大二時修習「程式設計與資料結構」，我對 C++ 的記憶體管理和演算法的效率優化有了初步的認識，理解了資料結構在程式設計中的關鍵角色。隨後的「作業系統」與「編譯器設計」更是讓我深深著迷。在「編譯器設計」課程中，我和組員們合作，使用 Lex 和 Yacc 實作了一個迷你語言的語法分析器（Parser），儘管只是個雛形，但能親手建構出將程式碼轉換為機器可執行指令的過程，讓我對軟體底層的運作有了更深刻的體會。而「軟體工程」這門課則讓我看到，一個複雜的軟體系統不僅僅是程式碼的堆疊，更需要嚴謹的規劃、版本控制與團隊協作。我們在課程專題中利用 Python 共同開發了一個簡單的學生社團管理系統，從需求分析、設計到實作，體會到良好軟體開發流程的重要性。雖然我缺乏參與研究室或產學合作的經驗，但這些課程專題和課後自學，都成為我技術能力的主要來源。

除了課業，我也曾投入於社團活動。大二、大三時，我曾擔任環境保護社團的幹部，負責籌劃校園環保宣導活動，並組織淨山行動。這讓我學會了團隊合作與溝通協調的重要性，也讓我對社會議題有更深的關懷。在社團中，我曾負責設計活動海報與文案，並協調各部門的合作，這些經驗都強化了我的人際互動與組織能力。

然而，隨著大學學業即將告一段落，我深深感受到自己在大學期間所學的深度仍有不足，尤其在實際應用與研究方面更是缺乏經驗。多年的打工生活雖然讓我學習到許多生活技能，但也讓我意識到，我渴望在專業領域有更深厚的發展，希望能轉換跑道，將大學所學的軟體知識進一步精進，並投入更專業的領域。因此，我希望能申請貴所，期待能有機會繼續深造，補足我在理論與實務上的不足，並為未來的職涯發展奠定更紮實的基礎。

### **讀書計畫**

進入國立交通大學資訊工程學系研究所深造，是我現階段最明確的目標。我希望能透過兩年的學習，補足自己在學術研究與專業技能上的不足，特別是深化我在程式語言、編譯器與軟體工程領域的知識，為未來進入軟體產業做準備。

**短期計畫（第一年：充實基礎與探索方向）**

在研究所的第一年，我將重心放在鞏固理論基礎與拓展專業知識上。我會積極選修相關的基礎課程，例如「高等軟體工程」、「進階編譯器設計」以及「演算法設計與分析」等。這些課程將幫助我建立更紮實的軟體開發流程、程式語言原理與效能優化的能力。我特別期待能在「高等軟體工程」中，學習到更多關於軟體架構設計與測試方法，並嘗試將這些理論應用到實際的專案中。我也會利用課餘時間，重新審視大學時期的程式語言課程內容，並透過線上資源或書籍，加強自己的程式實作能力，目標是能夠熟練運用 Python 和 Java 等語言來解決更複雜的問題。同時，我會主動旁聽系上舉辦的學術研討會或專題演講，希望能藉此接觸到不同教授的研究方向，並嘗試與研究室的學長姐交流，為未來選定指導教授和研究主題做初步的探索。儘管我在大學期間沒有正式的研究經驗，但我會積極學習研究方法，培養批判性思考與解決問題的能力。

**中期計畫（第二年：深入研究與論文撰寫）**

進入研究所第二年，在第一年所建立的基礎上，我將更聚焦於我的研究方向，並開始著手進行論文研究。我目前對軟體測試自動化、程式碼分析或特定領域程式語言設計等議題有較濃厚的興趣，這些都與我大學時所接觸的程式語言、編譯器及軟體工程領域緊密相關。我希望能找到一位研究方向契合的指導教授，並在他的指導下，逐步學習如何定義研究問題、蒐集文獻、設計實驗並分析結果。我會投入大量時間閱讀相關的學術論文，並嘗試撰寫研究報告，目標是在畢業前完成一份具有一定學術水準的碩士論文。此外，我也會考慮選修一些更進階或跨領域的課程，例如「形式化方法」或「分散式系統」，以拓寬我的知識廣度與深度，希望能將這些知識應用於我的研究課題中，為論文帶來更多元的觀點。同時，我也會積極參與系上的討論會，與同學們交流研究進度，學習如何清晰地表達自己的研究成果與想法。

**長期計畫（畢業後：投身軟體產業）**

完成碩士學業後，我希望能夠將所學應用於實務，進入軟體產業發展。我的目標是成為一名專業的軟體工程師，從事軟體開發、系統設計或品質保證等相關工作。我相信透過交大資工所紮實的訓練，我將具備更強大的問題解決能力、系統設計思維與團隊合作精神。我希望能貢獻所學，參與開發具創新性且穩定的軟體產品，為社會帶來實質的貢獻。未來，我也會持續關注資訊科技的最新發展，不斷精進自己的專業能力，並期望能在軟體工程領域佔有一席之地。
