## 自傳

我的名字是郭浩然，來自中央大學資訊工程學系。回顧大學四年，這是一段充滿挑戰與成長的旅程。我的學業成績，也就是修課平均，可能沒有達到非常亮眼的程度，大約是 2.67，但我相信這段經驗讓我對學習有了更深的體悟，也激發了我對資訊工程領域更堅定的熱情，尤其是在生醫訊號與影像分析這塊，有著很明確的興趣。

### 成長背景與學習啟蒙

從小，我就對數理邏輯有著一份難以言喻的熱情。高中時期，我有幸接受了數學奧林匹亞的培訓，那段日子讓我學會了如何拆解複雜的問題、從不同角度思考，並培養了嚴謹的邏輯推理能力。雖然最後沒有入選國手，但這段經歷為我的思考模式打下了良好的基礎，也讓我對「解決問題」這件事充滿了好奇與成就感。

進入中央大學資訊工程學系後，我對資訊科技廣闊的可能性感到興奮。然而，大學生活比我想像的更多元。我發現自己除了課業，也投入了大量心力在社團活動上，導致初期在學業上未能完全兼顧。但這段過程中，我逐漸理解到單純的理論知識與實際應用之間的距離，也開始思考如何將我的數理背景與資訊工程結合，去解決一些有意義的問題。大三那年，在修習「數位影像處理」與「生物資訊導論」時，我突然找到了方向——我被生醫領域透過資訊技術來提升人類健康的潛力深深吸引。從此，我開始積極地探索這個跨域的道路。

### 大學社團與領導經驗

大學四年，社團生活是我成長的重要舞台。我曾擔任系學會的活動組長，也積極參與了幾個大型校園活動的籌辦。其中，印象最深刻的是大二時我帶領團隊成功策劃並執行了系上的一年一度「資訊之夜」。這個活動動員了超過 50 位同學，並吸引了將近 400 名師生參與。從最初的企劃發想、預算編列、人員分工、廠商協調，到最後的現場執行與應變，每一個環節都充滿了挑戰。過程中，我們遭遇了贊助商臨時抽手、天氣驟變等突發狀況，但我學會了如何在壓力下保持冷靜，與團隊成員共同找出解決方案。例如，在贊助商問題上，我主動聯絡了超過十家企業，最終成功補齊了資金缺口。這段經驗讓我深刻理解到溝通協調、團隊合作以及危機處理的重要性，也大幅提升了我的領導能力和人際關係處理技巧。我學到了再困難的任務，只要大家目標一致、分工明確，總能找到出路。

此外，我也參與了校內的服務性社團，負責為偏鄉學童舉辦程式設計營隊。在營隊中，我需要將複雜的程式概念轉化為淺顯易懂的遊戲，讓孩子們能夠在玩樂中學習。這不僅考驗了我的教學與表達能力，更讓我體會到將資訊科技帶給更多人的意義。我記得有一個孩子在學會用 Scratch 寫出第一個小遊戲時的興奮表情，那份成就感讓我意識到，科技的影響力是如此巨大且深遠。

### 跨域探索與技術實踐

儘管我的成績不突出，但在找到生醫訊號與影像這個興趣點後，我努力透過修課和自學來彌補。在「數位影像處理」的課程專題中，我嘗試使用 Python 搭配 OpenCV 函式庫，對一些公開的醫學影像（例如胸腔X光片）進行基本的影像增強與邊緣偵測。雖然只是初步的嘗試，但這讓我理解到影像前處理對於後續分析的重要性。我也自主學習了生物醫學資料分析的基礎知識，嘗試利用 Pandas 處理一些公開的 EEG（腦電圖）訊號資料，進行基本的時頻分析，雖然成果還比較粗糙，但我初步認識了訊號處理的基本流程，例如如何濾波、如何去噪。

我沒有正式參與研究室，但透過這些課程專題和個人興趣專案，我開始接觸機器學習的基本概念。我嘗試用 TensorFlow 搭建過簡單的卷積神經網路（CNN）模型，去辨識貓狗圖片，這些都是從網路上的教學資源一步一步學習而來。雖然我的技術深度仍待加強，但我具備獨立學習新技術的熱情和嘗試的勇氣。我深信，我的高中數學奧林匹亞的邏輯訓練和大學社團的實作經驗，能讓我在未來的學術研究中，以更務實且具執行力的方式來解決問題。我學到的一點是，即使工具再強大，如果沒有清晰的問題定義和資料前處理，結果也只會是垃圾進垃圾出。

### 業界實習經驗與自我成長

在大三升大四的暑假，我在一家新創公司「智匯科技」擔任了為期三個月的初階軟體開發實習生。這是一個寶貴的機會，讓我能將課堂所學與業界實務結合。我的主要職責是協助開發公司的內部資料管理系統。我負責使用 SQL 撰寫查詢語句，從資料庫中提取特定數據，並學習使用 Git 進行版本控制，與團隊成員協同開發。雖然我的任務主要是協助性質，但這讓我真實體驗了軟體開發的流程，從需求分析到測試部署。我深刻理解到程式碼的品質、團隊合作的效率以及文件的重要性。

其中一個小專案是，我需要根據客戶需求，快速調整資料報表的呈現方式。我利用 Python 腳本結合 Pandas，快速處理並轉換了數十個資料表格的格式，將處理時間從人工的兩天縮短到幾個小時。這讓我對資料處理的效率與自動化有了更實際的體會，也讓我意識到軟體工具可以如何提升工作效率。這次實習也讓我學會了如何在時程壓力下完成任務，以及如何與不同背景的同事有效溝通。

### 申請動機與未來展望

回顧過去四年，我深知自己在學業表現上有所欠缺，但我從未放棄追求知識與自我提升。大學生涯的社團經歷培養了我的領導力、協作能力與解決問題的彈性；跨域的學習嘗試則讓我明確了未來的方向。我看到交大資工所擁有非常豐富的資源與頂尖的師資，特別是在人工智慧、機器學習以及與生醫結合的領域中，有許多令人振奮的研究成果。我對貴所陳彥宇教授在生物醫學訊號處理與機器學習的應用、以及王協芳教授在醫學影像分析方面的研究方向深感興趣。這些都與我個人希望結合資訊工程解決生醫問題的目標高度契合。

我渴望在貴所深造，系統性地學習更高深的理論知識與實作技巧，尤其是機器學習與深度學習在生醫訊號與影像分析上的應用。我期待能利用我的數理邏輯基礎與社團培養的實作精神，在貴所的學術環境中，努力彌補我過去在技術深度上的不足，並將我的跨域興趣轉化為具體的學術產出。我相信我的多元背景和強烈求知慾，能為貴所帶來不同的視野與貢獻。
