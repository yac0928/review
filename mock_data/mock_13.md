**自傳**

我是梁羿晨，來自國立臺灣大學資訊管理學系。從小，我就對複雜的系統運作與數據背後的意義充滿好奇。進入大學後，資管系的課程讓我得以接觸資訊科技的廣闊天地，更在學習過程中逐步聚焦於機器學習與智慧型計算的領域，這也成為我希望在研究所階段深入鑽研的核心方向。

回顧大學四年，我的學業成績雖然是 3.51，或許不算頂尖，但我深知知識的獲取不只在課堂，更在於實際應用與動手實作。大三時，我積極爭取進入系上的「智能金融實驗室」，參與了「基於深度學習的股價趨勢預測模型」專案。在這個專案中，我負責數據前處理與模型建構，主要使用 Python 搭配 TensorFlow 框架。我們從公開的財報與新聞資料中爬取數據，並嘗試將多種時間序列模型（如 LSTM、GRU）應用於股價預測。這段經歷讓我深刻體會到金融數據的複雜性，例如資料的雜訊多、非線性強，以及市場情緒等非結構化資訊的難以量化。我學會了如何進行特徵工程，將原始數據轉換為模型能有效學習的特徵，也明白了單一模型無法涵蓋所有變數，因此我們也試著融合多模態數據來提升預測準確度，雖然結果仍有其限制，但過程中對模型調校與效能評估有了更實際的理解。這次經驗讓我對於如何將前沿的機器學習技術應用於高度變動的財金領域充滿了興趣，也激發了我深入探索智慧型計算的決心。

除了學術研究，我也曾於一家新創金融科技公司「智匯資本」實習。我在其中擔任資料分析助理，主要職責是協助開發內部資料視覺化工具，將客戶的交易行為與投資組合表現數據化。這個經驗讓我實際接觸到金融業界如何運用數據來驅動決策。我使用 SQL 查詢資料庫，並透過 Tableau 製作互動式儀表板，為業務團隊提供即時的市場洞察。這段實習讓我意識到，即便擁有技術，也需要配合實際的業務需求與應用場景，才能發揮最大的價值，這讓我對軟體開發的「產品導向思維」有了初步的認識。

我認為自己在財金領域的專業背景，能為資工領域帶來不同的視角。在處理金融相關的數據時，我對其背後的經濟意義與市場機制有一定程度的理解，這有助於我在機器學習模型建構時，能更合理地選擇特徵、解釋結果，並提出具有實務價值的應用方向。

我之所以選擇貴所，是因為交通大學資訊工程學系在機器學習與智慧型計算領域的卓越聲譽與豐沛資源。貴所不僅擁有許多在相關領域深耕的頂尖教授，更有許多與產業緊密合作的機會。我渴望能在貴所的引導下，系統性地學習更深奧的演算法理論、先進的深度學習模型，並將我過去在財金領域的知識與資工技術完美結合，期待未來能在智慧金融或智慧決策的應用中貢獻一己之力。
