# 國立交通大學資訊工程學系碩士班 推甄資料

## 自傳

## 我的背景與成長

我叫洪育翔，來自國立中山大學資訊工程學系。從小，家裡的經濟狀況就比較吃緊，這使得我從高中畢業後，就開始利用課餘時間打工，希望能為家裡減輕負擔，也支應自己的學費與生活開銷。這段歷程雖然辛苦，卻也讓我比同儕更早接觸社會，學習到許多課本以外的寶貴經驗。

選擇資訊工程這條路，其實是源於對解決問題的興趣。我一直覺得科技有種魔力，能讓複雜的事情變簡單，甚至創造出新的可能性。雖然起初對程式碼的世界有點陌生，但每當我成功寫出一個小程式，或是讓某個專案動起來時，那種成就感都讓我更加確定，這是我想投入的領域。

## 大學生活與學業表現

在大學期間，我的生活重心除了課業，很大一部分就是各式各樣的打工。我坦承這對我的學業成績確實造成了影響，我的學業成績約為 3.15。儘管如此，我始終盡力在重要的專業課程上投入心力。我修習了包括「資料結構與演算法」、「計算機網路」、「作業系統」、「嵌入式系統」等核心課程。

在大三時，我在「嵌入式系統」的課程專題中，嘗試與同學合作開發一個簡易的智慧居家監控系統。我們利用 **ESP32** 微控制器作為核心，透過 Wi-Fi 模組收集溫濕度資料，並將數據上傳到簡易的雲端平台。雖然這是一個基礎專案，但在設計硬體介面、撰寫韌體程式（使用 **C++** 語言）、以及處理無線通訊協定方面，都讓我對物聯網的運作原理有了初步的認識。透過這次經驗，我具體了解到從感測器到網路傳輸再到後端處理的整個流程，並體會到穩定通訊的重要性。

另外，我也對「計算機網路」課程中各種網路協定、封包傳輸、以及無線網路技術的原理感到特別著迷。儘如時間有限，我也利用課後時間透過 **Wireshark** 等工具，觀察並分析網路封包，試圖更深入地理解理論知識在實際中的展現。

## 打工經驗與社會參與

大學四年，我的足跡遍佈各式服務業。我曾在「茶湯會」擔任飲料店店員，負責從點餐、調製飲品、備料到維持店面整潔等所有事務。這份工作讓我學習到如何在高壓環境下保持效率與精準，也體會到耐心與良好溝通對於顧客服務的重要性。另外，我也做過「7-ELEVEN」的晚班店員，負責補貨、結帳、以及處理顧客的各式需求。這讓我學會了更細心觀察、隨機應變，並在面對不同情境時，快速找到解決方案。這些非技術性工作，或許看似與資工領域無關，卻磨練了我的責任感、抗壓性以及與人協作的能力。我深知，這些「軟實力」在任何領域都是不可或缺的。

除了打工，我也利用時間參與了「公民科技參與 (g0v)」社群的一些活動。雖然我沒有直接參與程式開發，但我會關注他們的專案進展，並在一些開放資料專案的初期，協助進行資料的初步整理與文件撰寫。有時候也會參與線下聚會，聽取前輩們分享如何運用科技解決社會問題。這份經驗讓我跳脫了純技術的框架，開始思考資訊技術如何能夠真正地服務社會、提升公共福祉。我特別被其中關於物聯網應用於環境監測或智慧交通的討論所吸引，讓我對物聯網的實際應用面產生了更濃厚的興趣。我了解到，科技不只是一串程式碼或一套系統，它背後承載的是解決實際問題的可能。

## 申請動機與未來展望

回顧過去四年，我深知這些非技術性打工經驗，讓我的人生歷練相對豐富，但也讓我深刻感受到技術能力的不足，難以在這個快速變遷的時代站穩腳步。面對未來，我渴望能有機會更專注於學術研究與技術精進，因此決定申請貴所，希望能轉換跑道，往更深更專業的技術領域發展。

國立交通大學資訊工程學系在物聯網通訊與無線網路領域的卓越聲譽，以及豐沛的研究資源，一直是我所景仰的。貴所多位老師在無線通訊、物聯網安全、感測網路等領域的研究成果斐然，這與我個人在大學期間對「物聯網通訊」及「無線網路」的興趣高度契合。我特別希望能投入在如何優化物聯網裝置的能源效率、提升無線資料傳輸的可靠性與安全性等方向進行學習與研究。我渴望能在貴所完善的課程體系下，補足我過去在基礎理論與實作經驗上的短板，並能有機會參與到尖端的研究專案中。

我深信，我過去獨立解決問題、快速學習以及堅韌的特質，加上對資訊工程的熱情與對未來明確的規劃，能夠讓我在交大資工所攻讀碩士期間，展現出高度的學習潛力與投入。我期望未來能成為物聯網架構師或無線網路工程師，將所學貢獻於社會，甚至能將公民科技的理念融入其中，創造出更具影響力的應用。

## 交大資工所碩士班 讀書計畫

### 短期目標 (入學前至碩一上學期)

-   **基礎能力補強：** 針對大學時期可能因打工而未充分掌握的基礎知識進行複習與深化。主要複習科目包括：資料結構與演算法、作業系統、計算機網路，並透過LeetCode或Hackerrank等平台加強程式解題能力。
-   **英語能力提升：** 由於未來會接觸大量英文學術論文，我會加強英文閱讀能力，並練習學術寫作，目標是能夠流暢閱讀專業文獻。
-   **研究方向探索：** 在入學前會主動瀏覽貴所教授的研究領域與論文，鎖定數個與「物聯網通訊」及「無線網路」相關的實驗室。入學後會積極與指導教授面談，深入了解各實驗室的研究主題與計畫，以確立自己的碩士論文方向。我特別對低功耗廣域網路 (LPWAN) 如 LoRa 或 NB-IoT 的應用與優化感興趣。

### 中期目標 (碩一下學期至碩二上學期)

-   **專業課程學習：** 根據選定的研究方向，選修相關的專業課程。這可能包括：「高等網路協定」、「物聯網系統設計與應用」、「無線感測網路」、「嵌入式系統進階」或「網路安全」等。我也會考慮修習一些機器學習相關的課程，因為物聯網數據的分析和處理，常常會用到相關技術。
-   **專題研究參與：** 積極投入實驗室的專案研究。我期望能參與到實際的物聯網原型系統開發中，例如：設計並實作基於 **Raspberry Pi** 或 **Arduino** 的智慧感測節點，搭配 **LoRaWAN** 或 **Zigbee** 等無線技術進行數據採集與傳輸實驗，並針對傳輸效率、功耗或安全性等議題進行探討。
-   **學術文獻研讀：** 定期閱讀領域內的頂級會議與期刊論文，追蹤最新研究趨勢，並學習研究方法與論文寫作技巧，為碩士論文打下堅實基礎。

### 長期目標 (碩二下學期與畢業後)

-   **碩士論文完成：** 依照規劃，在指導教授的帶領下，完成具有創新性與實作價值的碩士論文。我會爭取在國內外研討會或期刊發表研究成果，以提升自己的學術視野與能力。
-   **技術應用與職涯發展：** 畢業後，我期望能將在交大資工所學到的專業知識與研究經驗，應用於實際產業。我對物聯網解決方案開發、無線網路架構設計、或是與公民科技結合的智慧城市應用有濃厚興趣。期望能成為一名能夠將理論付諸實踐、解決實際問題的專業技術人才。
-   **持續學習與回饋：** 我會持續關注資訊科技的發展，不斷學習新的技術與知識。同時，也期望未來能有機會回饋社會，貢獻所長，特別是在公民科技領域，希望能夠將學術研究的成果，轉化為對社會有正面影響的實際應用。
