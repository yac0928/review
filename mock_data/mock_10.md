## 自傳

我的名字是賴品睿，畢業於國立中興大學生物醫學工程學系，並雙主修心理學系。我從小就對人體的奧秘與心智的運作充滿好奇，這份好奇心引導我進入了生醫工程領域。大學期間，我逐漸發現，要更深入地理解生命現象，單靠傳統的生物醫學知識是不夠的，必須結合資訊科學的強大分析能力。特別是在處理生醫訊號與醫學影像時，我看到了資訊科技如何能將複雜的數據轉化為有價值的臨床洞察，這讓我對跨域學習產生了濃厚的興趣。

我的大學生涯可以說是一個不斷探索與融合的過程。在生物醫學工程學系，我學習了生理學、解剖學等基礎醫學知識，也接觸了生物醫學訊號處理與醫學影像學。這些課程讓我認識到，無論是心電圖 (ECG) 還是腦波圖 (EEG)，甚至是核磁共振 (MRI) 影像，其背後都蘊藏著大量的資訊等待被解讀。大三時，我選修了「數位影像處理」與「機器學習導論」等課程，這成為我跨足資工領域的一個重要轉折點。在數位影像處理的課程專題中，我嘗試使用 Python 和 OpenCV 函式庫，開發了一個基於邊緣檢測的細胞計數輔助系統，雖然成果還很基礎，但親手將演算法應用於實際影像的體驗讓我印象深刻。

同時，心理學的雙主修則給了我一個獨特的視角。心理學讓我學會如何從人類行為、認知模式和知覺層面去思考問題，這對於設計以人為本的醫療科技、理解醫病互動，甚至在未來開發更符合使用者經驗的醫學影像判讀系統時，都提供了寶貴的啟發。雖然學業成績 3.41 未臻頂尖，這份成績反映了我花費大量的時間在不同領域的知識探索與整合上，並嘗試將兩個看似獨立的學科結合應用。

儘管我沒有正式的研究室經驗或產學合作機會，但大學期間我積極透過修課與自學來彌補。在「生物醫學訊號處理」課程中，我學習了如何使用 MATLAB 處理並分析各類生理訊號，並在期末專題中，針對一段癲癇腦波數據進行了初步的頻譜分析與特徵提取，我從中理解了濾波、傅立葉變換這些基本概念的重要性。此外，我也自學了深度學習的基礎知識，透過 Coursera 的線上課程與 Keras 函式庫，嘗試過用卷積神經網路 (CNN) 對一些簡單的醫學影像（例如胸腔 X 光片）進行異常檢測的練習，初步體會到模型訓練與資料預處理的挑戰。這些經驗讓我對資料科學和人工智慧在醫療領域的巨大潛力有了更具體的認識，也更加堅定我未來投入此領域的決心。

我深知要將生醫領域的專業知識與尖端資訊科技完美結合並不容易，這也是我決定申請貴所資工所的原因。交大資工所不僅在資訊工程領域享有盛譽，更擁有許多在機器學習、深度學習和數據科學方面有著深厚研究的教授。我希望能在這裡，系統性地學習更扎實的資訊科學基礎，並在教授的指導下，將我過去的生醫與心理背景轉化為獨特的優勢，為未來的研究奠定基石。我相信我的跨域背景能為研究帶來不同的觀點與可能性。

## 讀書計畫

我申請交通大學資訊工程學系碩士班，是希望在資訊工程的堅實基礎上，深耕生醫訊號處理與醫學影像分析的領域。我的目標是透過人工智慧與機器學習技術，提升醫療診斷的精準度與效率。基於我過往的跨域學習背景，我將我的讀書計畫分為短期、中期與長期三個階段。

### 短期計畫（碩一上學期）

在碩一上學期，我的首要任務是補強資訊工程的核心知識，特別是與機器學習和數據分析相關的理論基礎。我將積極選修貴所開設的「機器學習特論」、「深度學習」或「高等演算法」等課程，確保我能跟上研究進度。同時，我會努力學習更進階的程式設計技巧，熟悉 Python 在科學計算上的應用，例如 NumPy、Pandas 等數據處理工具，以及 TensorFlow 或 PyTorch 等深度學習框架。
我的目標是透過這些課程與自學，將我過去相對零散的資訊應用知識，系統化地整合起來，建立一套完整的思維框架。我也會積極參與實驗室的專題討論，盡快融入研究環境。

### 中期計畫（碩一下學期至碩二上學期）

進入碩一下學期後，我將更專注於我的研究方向——生醫訊號與醫學影像的智慧分析。我計劃選修如「醫學影像處理」、「生物資訊」或「計算生物學」等與我興趣相關的課程。我希望能結合我生醫工程的背景，將所學應用於實際的醫療數據。例如，我對利用深度學習模型進行病灶的自動識別、醫學影像的分割與重建，以及心電圖、腦波等生理訊號的異常模式檢測非常感興趣。

我將主動與指導教授討論，尋找合適的研究主題。我期待能參與到教授的專案中，透過實戰累積經驗。例如，我會嘗試運用不同模型（如 U-Net, ResNet 等）處理特定醫學影像數據集，進行圖像分類或語義分割的任務，並學會評估模型的性能指標。我也會花時間研究相關的學術論文，了解最新的研究進展與挑戰，並嘗試提出自己的初步想法與解決方案。同時，我會利用我心理學雙主修的背景，思考使用者介面設計、醫護人員對 AI 輔助系統的接受度等議題，讓我的研究更具人文關懷與應用價值。

### 長期計畫（碩二下學期）

到了碩二下學期，我的目標是完成碩士論文並順利畢業。我會根據中期的研究成果與發展，深化我的研究主題，並將成果撰寫成論文。在此階段，我將專注於提升研究的深度與廣度，例如嘗試創新演算法，或者將跨領域的知識更緊密地整合。我希望我的論文能夠在生醫資訊領域有所貢獻，無論是提出新的診斷輔助方法，或是優化現有的數據分析流程。

畢業後，我希望能夠將我在交大資工所學到的知識與技能，應用於醫療科技產業，特別是與人工智慧醫療、精準醫療相關的研發工作。我深信，在交大資工所的栽培下，我能成為一名具備跨領域視野且能實際解決問題的工程師，為推動台灣乃至全球的智慧醫療發展貢獻一份心力。
